uint8_t adc_get_active_channel ( void  )

Gibt die laufende Nr. des Channels zurueck, der aktuell ausgewertet wird. 0: erste registrierter Channel, 1: zweiter registrierter Channel usw. 255: derzeit wird kein Channel ausgewertet (= Konvertierung fertig)
