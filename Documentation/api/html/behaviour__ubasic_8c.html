uint8_t behaviour_result ( Behaviour_t *  behaviour)

Rueckgabe ob das zuletzt aufgerufene Verhalten erfolgreich war oder nicht; in Remotecall wird dazu via ubasic_push_value aufgerufen und das Ergebnis nach result_behav gebracht; in Ubasic selbst kann mit dieser Routine das Ergebnis ermittelt werden.

Rückgabe:
True falls das Verhalten erfolgreich war sonst nicht, je nach Kontext des Verhaltens
