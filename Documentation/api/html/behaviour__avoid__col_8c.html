Dokumentation der Funktionen

void bot_avoid_col_behaviour ( Behaviour_t *  data)

Passt auf, dass keine Kollision mit Hindernissen an der Front des Roboters geschieht. Diese Funktion ist nur ein Dummy-Beispiel, wie eine Kollisionsvermeidung aussehen koennte. Hier ist ein guter Einstiegspunkt fuer eigene Experimente und Algorithmen!
