void remotecall_display ( void  )

Displayhandler fuer RemoteCall-Display.

Das Display zeigt eine Liste aller verfuegbaren Verhalten an, durch die mit den Tasten "Stopp" (nach unten) und "Pause" (nach oben) geblaettert werden kann. Das Zeichen '>' markiert dabei eine ausgewaehlte Zeile. Bei Verhalten ohne Parameter wird das ausgewaehlte Verhalten mit der Taste "Play" gestartet. Benoetigt ein Verhalten Parameter, waehlt "Play" das Verhalten aus und schaltet auf die Parametereingabe um. Ueber die Zifferntasten koennen nun alle Parameter eingegeben werden, die Taste "Play" schliesst dabei jeweils die Eingabe ab und schaltet zum naechsten Parameter weiter. Mit "Stopp" laesst sich die aktuelle Eingabe loeschen und korrigieren. Ein Minus-Zeichen ('-') fuer negative Parameterwerte laesst sich mit der Taste "11" eingeben. Nach dem letzten Parameter startet "Play" das Verhalten.
