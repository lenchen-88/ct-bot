Konfig-Optionen fuer Dateisystem BotFS.

Autor:
Timo Sandmann (mail@timosandmann.de)
Datum:
17.10.2010

Konfiguration derzeit: max. Groesse des Volumes: 32 MByte max. Anzahl an Dateien: 256 max. Dateinamenlaenge: 116 Zeichen max. Dateigroesse: ~16 MByte (2^15 - 1 * 512 Byte)

moechte man da etwas aendern, muss man die Groesse der Datentypen beachten / anpassen!
