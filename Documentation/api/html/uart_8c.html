ISR ( USART_RXC_vect  )

Interrupthandler fuer eingehende Daten Empfangene Zeichen werden in die Eingabgs-FIFO gespeichert und warten dort.

Definiert in Zeile 91 der Datei uart.c.

Benutzt True.

ISR ( USART_UDRE_vect  )

Interrupthandler fuer ausgehende Daten Ein Zeichen aus der Ausgabe-FIFO lesen und ausgeben. Ist das Zeichen fertig ausgegeben, wird ein neuer SIG_UART_DATA-IRQ getriggert. Ist die FIFO leer, deaktiviert die ISR ihren eigenen IRQ.

Definiert in Zeile 105 der Datei uart.c.

Benutzt fifo_t::count und True.

void uart_init ( void  )

Initialisiert den UART und aktiviert Receiver und Transmitter sowie die Interrupts. Die Ein- und Ausgebe-FIFO werden initialisiert. Das globale Interrupt-Enable-Flag (I-Bit in SREG) wird nicht veraendert.

Definiert in Zeile 47 der Datei uart.c.

Benutzt fifo_init(), inbuf, UART_BUFSIZE_IN, outbuf, UART_BUFSIZE_OUT und __builtin_avr_cli().

Wird benutzt von ctbot_init().
