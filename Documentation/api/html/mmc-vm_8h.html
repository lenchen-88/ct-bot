Makro-Dokumentation

#define mmc_fopen (   filename)    mmc_fopen_P(PSTR(filename))

Oeffnet eine Datei im FAT16-Dateisystem auf der MMC / SD-Card und gibt eine virtuelle Adresse zurueck, mit der man per mmc_get_data() einen Pointer auf die gewuenschten Daten bekommt. Das Ein- / Auslagern macht das VM-System automatisch. Der Dateiname muss derzeit am Amfang in der Datei stehen. Achtung: Irgendwann muss man die Daten per mmc_flush_cache() oder mmc_page_write_back() zurueckschreiben!

Parameter:
filenameDateiname als 0-terminierter String
Rückgabe:
Virtuelle Anfangsadresse der angeforderten Datei in Byte, 0 falls Fehler

Definiert in Zeile 120 der Datei mmc-vm.h.

Wird benutzt von create_emu_mini_fat_file() und delete_emu_mini_fat_file().
