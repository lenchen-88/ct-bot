uint8_t init_eeprom_man ( uint8_t  init)

Diese Funktion initialisiert die eeprom-emulation. Sie sorgt fuer die Erstellung der eeprom.bin, falls nicht vorhanden und erstellt ueber eine Hilfsfunktion eine Adress- konvertierungstabelle fuer die EEPROM-Adressen, wenn die benoetigten Daten vorliegen. Statusinformationen werden ueber DEBUG_INFO angezeigt.

Parameter:
initgibt an, ob das EEPROM mit Hilfer einer eep-Datei initialisiert werden soll (0 nein, 1 ja)
Rückgabe:
0: alles ok, 1: Fehler

Definiert in Zeile 511 der Datei eeprom_pc.c.

Wird benutzt von ctbot_init_low_1st() und hand_cmd_args().
