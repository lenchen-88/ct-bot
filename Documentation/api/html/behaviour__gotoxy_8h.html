#define USE_GOTO_POS_XY

bot_gotoxy() beruecksichtig keine waehrend der Fahrt aufgetretenen Fehler, daher ist die Endposition nicht unbedingt auch die gewuenschte Position des Bots. Das komplexere Verhalten bot_goto_pos() arbeitet hier deutlich genauer, darum werden jetzt alle bot_gotoxy()-Aufrufe auf das goto_pos-Verhalten "umgeleitet", falls dieses vorhanden ist. Moechte man das jedoch nicht und lieber weiterhin das alte bot_gotxy-Verhalten, deaktiviert man den Schalter USE_GOTO_POS_XY ein paar Zeilen unter diesem Text, indem man ihm // voranstellt.
