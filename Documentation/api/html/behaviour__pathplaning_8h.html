Ausführliche Beschreibung

Wave-Pfadplanungsverhalten.

Eine niedrigaufloesende Map wird ueber die hochaufloesende gelegt und auf dieser folgende Schritte ausgefuehrt: -Innerhalb des wirklich benutzten Mappenbereiches wird jede Zelle durchlaufen und falls der Durchschnittswert der Hochaufloesenden Map < 0 ist (Hinderniswert) hier eingetragen mit Wert 1 -anzufahrende Zielposition erhaelt Mapwert 2 -ausgehend von Zielposition laeuft eine Welle los bis zum Bot-Ausgangspunkt, d.h. von 2 beginnend erhaelt jede Nachbarzelle den naechst hoeheren Zellenwert -wird als ein Nachbar die Botposition erreicht, wird der Pfad zurueckverfolgt und immer die Zelle mit kleinerem Wert gewaehlt; die Zellenkoordinaten werden als Weltkoordinaten auf den Stack gelegt und koennen nun abgefahren werden
